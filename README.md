## 🚀 部署步骤

### 1. 创建 GitHub 仓库
```bash
# 在 GitHub 上创建一个新的公共仓库
# 例如：https://github.com/yourusername/yourusername.github.io
```

### 2. 推送代码到 GitHub
```bash
# 添加远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/yourusername/yourusername.github.io.git

# 推送到 GitHub
git push -u origin master
```

### 3. 启用 GitHub Pages
1. 进入仓库的 **Settings** 页面
2. 点击左侧的 **Pages**
3. 在 **Source** 下选择：
   - Branch: `master` (或 `main`)
   - Folder: `/ (root)`
4. 点击 **Save**

### 4. 访问你的主页
等待 1-2 分钟后，访问：
```
[https://yourusername.github.io](https://asteroidf4ll.github.io/Vibe-Coding/)
```
